## 파일 구조
```
kindergarten_rag_chatbot/

│   ├── .env.example               # 환경 변수 예시
│   └── .gitignore                 # Git 제외 파일
│
├── 📖 문서 (3개)
│   ├── README.md                  # 프로젝트 개요
│   ├── GUIDE.md                   # 상세 사용 가이드 (15페이지)
│   └── QUICKSTART.md              # 5분 빠른 시작
│
├── 🔧 유틸리티 (2개)

└── 📁 데이터 폴더 (2개)
    ├── uploads/                   # 업로드된 참고 문서
    └── saved_evaluations/         # 저장된 평가문
```

---

## 🚀 두 가지 버전

### 1️⃣ 기본 버전 (app.py)
- **특징**: 빠르고 간단, 즉시 사용 가능
- **장점**: 설치가 쉽고 실행이 빠름
- **추천 대상**: 처음 사용하는 교사, 빠른 평가 작성이 필요한 경우
- **필요 패키지**: streamlit, openai, python-dotenv

### 2️⃣ 고급 버전 (app_advanced.py)
- **특징**: RAG 기반, 문서 업로드 및 검색
- **장점**: 
  - PDF, PPTX, TXT 파일 업로드
  - 업로드한 문서에서 유사 내용 자동 검색
  - OpenAI Embeddings로 의미 기반 검색
- **추천 대상**: 기존 평가 사례를 활용하고 싶은 경우
- **추가 패키지**: PyPDF2, python-pptx

## 📝 주요 기능

### ✅ 자동 평가문 생성
- 아이의 관찰 내용 입력만으로 전문적인 평가문 자동 생성
- 누리과정 5개 영역 자동 반영
- 따뜻하고 전문적인 교사 말투

### ✅ 발달적 근거 제시
- 각 행동의 발달적 의미 설명
- 참고한 발달 영역 명시
- 교사의 전문성을 드러내는 해석

### ✅ 대화형 수정
- "더 따뜻하게 써주세요"
- "사회관계 부분을 더 구체적으로"
- 실시간 피드백 반영

### ✅ 가정 연계 제안
- 가정에서의 지원 방안 자동 생성
- 긍정적이고 실천 가능한 제안

## 💰 비용 안내

### OpenAI API 사용료
- **GPT-4**: $0.03 (입력) + $0.06 (출력) / 1K 토큰
- **Embeddings** (고급 버전): $0.0001 / 1K 토큰

### 예상 비용
- **평가문 1개**: 약 130~260원
- **월 30개 작성**: 약 4,000~8,000원
- **학기당 60개**: 약 8,000~16,000원

💡 학급당 20명 기준 학기말 평가 비용: 약 3,000~5,000원

## 🎯 사용 시나리오

### 시나리오 1: 학기말 평가 작성
**상황**: 20명 반 학기말 평가 작성
**사용 버전**: 기본 버전
**소요 시간**: 아이 1명당 3~5분
**총 시간**: 1~2시간 (vs 기존 4~6시간)

### 시나리오 2: 일관된 평가 스타일
**상황**: 작년 평가 사례를 참고하여 일관성 유지
**사용 버전**: 고급 버전
**방법**: 
1. 작년 평가 PPTX 업로드
2. AI가 스타일 학습
3. 일관된 톤으로 생성

### 시나리오 3: 신입 교사 지원
**상황**: 처음 평가문 작성하는 신입 교사
**사용 버전**: 기본 버전 or 고급 버전
**장점**:
- 평가문 구조 학습
- 전문 용어 사용법 습득
- 발달적 해석 방법 이해

## 🔒 보안 및 개인정보

### ⚠️ 주의사항
1. **OpenAI API 키는 절대 공유하지 마세요**
   - 개인 계정에 비용이 청구됩니다
   
2. **아이의 개인정보 보호**
   - 성명은 가명 사용 권장
   - 업로드한 문서는 사용 후 삭제
   
3. **데이터 저장 위치**
   - OpenAI 서버에 일시적으로 저장 (API 정책)
   - 로컬 컴퓨터의 `saved_evaluations/` 폴더

### ✅ 권장 사항
- API 키는 `.env` 파일에 저장
- 업로드 문서는 개인정보 제거 후 사용
- 저장된 평가문은 정기적으로 백업

## 📚 누리과정 5개 영역

프로그램에 내장된 발달 지식:

1. **신체운동·건강**
   - 대근육, 소근육 발달
   - 건강한 생활습관

2. **의사소통**
   - 듣기, 말하기
   - 읽기, 쓰기

3. **사회관계**
   - 자기존중, 정서인식
   - 또래관계, 협력

4. **예술경험**
   - 탐색, 표현
   - 감상, 음률

5. **자연탐구**
   - 탐구과정, 생명존중
   - 수학적·과학적 탐구

각 영역별 만 3세~만 5세 발달 특성 포함
