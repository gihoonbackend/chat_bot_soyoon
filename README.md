# 유아발달상황 종합평가 작성 도우미 🎓

## 프로젝트 소개
유치원 교사를 위한 RAG(Retrieval-Augmented Generation) 기반 챗봇입니다. 
아이의 관찰 내용을 입력하면 누리과정 5개 영역(신체운동건강, 의사소통, 사회관계, 예술경험, 자연탐구)을 포함한 따뜻한 평가문을 자동으로 생성합니다.

## 주요 기능
✅ **AI 기반 평가문 자동 생성**: 관찰 내용을 입력하면 전문적인 평가문 작성
✅ **5개 발달 영역 통합**: 누리과정 기반 체계적인 평가
✅ **근거 제시**: 각 평가의 발달적 근거와 참고 자료 표시
✅ **대화형 수정**: 생성된 평가문을 대화를 통해 수정 가능
✅ **문서 저장**: 작성된 평가문을 텍스트 파일로 저장

## 기술 스택
- **Frontend**: Streamlit
- **AI Model**: OpenAI GPT-4
- **Language**: Python 3.10+

## 설치 방법

### 1. 저장소 클론
```bash
git clone <repository-url>
cd kindergarten_rag_chatbot
```

### 2. 가상환경 생성 및 활성화
```bash
python -m venv venv
source venv/bin/activate  # Linux/Mac
# 또는
venv\Scripts\activate  # Windows
```

### 3. 필요한 패키지 설치
```bash
pip install -r requirements.txt
```

## 실행 방법

### 1. Streamlit 앱 실행
```bash
streamlit run app.py
```

### 2. 웹 브라우저에서 접속
자동으로 브라우저가 열리며, 기본 주소는 `http://localhost:8501` 입니다.

## 사용 방법

### Step 1: API 키 입력
1. 왼쪽 사이드바에서 OpenAI API 키를 입력합니다
2. API 키는 [OpenAI 플랫폼](https://platform.openai.com/api-keys)에서 발급받을 수 있습니다

### Step 2: 문서 업로드 (선택사항)
1. 참고할 평가 사례나 발달 자료를 업로드합니다
2. 지원 형식: PDF, TXT, HWP, PPTX

### Step 3: 아이 정보 입력
1. **아이 이름**: 예) 고도율
2. **나이**: 만 3세 ~ 만 7세 선택
3. **학기**: 1학기 또는 2학기 선택
4. **관찰 내용**: 구체적인 행동 관찰 내용 입력

예시:
```
하츄핑을 그리고 잘라서 그걸로 친구와 역할놀이함. 
경험한 일에 대해 기억하고 교사나 친구들에게 이야기함. 
도전하는 음식이 많고 골고루 먹으며 남기지 않고 먹는 모습이 많아짐.
```

### Step 4: 평가문 생성 및 수정
1. "✍️ 평가문 생성" 버튼 클릭
2. AI가 생성한 평가문 확인
3. 필요시 추가 질문이나 수정 요청 입력
4. "📋 복사" 또는 "💾 저장" 버튼으로 결과 저장

## 평가문 작성 기준

### 1. 작성 원칙
- **따뜻하고 상냥한 교사 말투** 사용
- 아이의 **행동 관찰 → 발달적 의미 해석 → 교사의 전문성** 순서로 작성
- **자연스러운 문맥 연결**로 읽기 쉽게 구성

### 2. 필수 포함 영역
1. **신체운동건강**: 대근육, 소근육, 건강한 생활습관
2. **의사소통**: 듣기, 말하기, 읽기, 쓰기
3. **사회관계**: 자기존중, 정서인식, 또래관계, 협력
4. **예술경험**: 탐색, 표현, 감상, 음률
5. **자연탐구**: 탐구과정, 생명존중, 수학적·과학적 탐구

### 3. 구조
- **도입**: 아이의 전반적인 성장과 모습
- **본문**: 각 영역별 구체적 관찰과 발달적 의미
- **마무리**: "~의 2학기 생활도 응원합니다!" + 가정 연계 지원 방안

## 예시 출력

### 입력
- 이름: 고도율
- 나이: 만 3세
- 관찰: 하츄핑을 그리고 잘라서 그걸로 친구와 역할놀이함. 경험한일에대해 기억하고 교사나 친구들에게 이야기 함, 도전하는 음식이 많고 골고루 먹으며 남기지 않고 먹는 모습이 많아짐.

### 출력
```
항상 밝고 긍정적인 에너지를 가지고 생활하는 도율이는 다양한 경험을 통해 
즐겁게 성장하고 있습니다. 특히, 하츄핑을 직접 그리고 오려서 친구들과 
역할놀이를 하는 모습이 인상적이며, 이를 통해 창의력과 상상력을 자연스럽게 
펼쳐 나가고 있습니다...

(중략)

1학기 동안 유치원 생활을 즐기며 적극적으로 참여한 도율이의 2학기 생활도 
응원합니다! 가정에서도 다양한 놀이와 대화를 통해 도율이가 더욱 풍부한 
경험을 쌓고, 자신의 생각을 자유롭게 표현할 수 있도록 따뜻한 관심을 
가져주시면 좋겠습니다. 😊
```

